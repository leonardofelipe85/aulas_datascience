# 📊 Análise de Inteligência de Mercado — Superstore

Projeto desenvolvido no LAB365 | SENAI/SC com foco em análise exploratória de dados utilizando Python, Pandas e Matplotlib.

---

## 🎯 Objetivo

O projeto tem como objetivo analisar dados comerciais de uma rede varejista, identificando padrões de vendas, comportamento do lucro, impacto dos descontos e desempenho regional.

Além da análise visual, também foi realizado um processo completo de tratamento e higienização dos dados.

---

## 🛠 Tecnologias Utilizadas

- Python
- Pandas
- Matplotlib
- Openpyxl
- VS Code

---

## 📂 Estrutura do Projeto

```bash
Projeto_superstore/

├── analise_superstore.py
├── SampleSuperstore.csv
├── relatorio_final.pdf
├── dataset_tratado.xlsx
├── resumo_gerencial.xlsx

└── graficos/
```

---

## 📈 Principais Insights

- Descontos elevados impactam negativamente o lucro;
- A região West apresentou o maior lucro acumulado;
- O faturamento apresentou crescimento contínuo entre 2014 e 2017;
- O tratamento correto dos dados foi essencial para evitar distorções analíticas.

---

## ▶️ Como Executar

Instale as bibliotecas necessárias:

```bash
pip install pandas matplotlib openpyxl
```

Execute o projeto:

```bash
python analise_superstore.py
```

---

## 📊 Visualizações Geradas

O projeto gera automaticamente:

- Vendas por categoria;
- Lucro por categoria;
- Vendas por região;
- Impacto do desconto no lucro.

Os gráficos são salvos automaticamente na pasta `graficos/`.
